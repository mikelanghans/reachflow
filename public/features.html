AI-written performance summaries generated from live campaign data. Shareable link, PDF export, branded header. Takes 4 seconds.

Review queue Ahead of Syndie
Toggle review mode per campaign. Every outgoing message appears for approval before sending. Approve as-written, edit, or reject. Bulk approve available.
Pain solved: AI-generated messages going out under a client's LinkedIn without a QA step is a liability. Review mode is the safety net.

Per-client audit log New
Complete activity history per client — every message sent, reply received, meeting booked, stage change. Filterable by event type, exportable to CSV.

Auto-detects opt-out language in replies. Adds contacts to suppression list. Disables messaging for suppressed contacts. GDPR/CAN-SPAM compliant. CSV export.

Technology stack
What powers ReachFlow
Every service we use, what it does, and why we chose it.
⚛
React + Vite
Frontend framework
The entire UI — 5,000+ lines of component code. Vite handles bundling and hot reload during development. Deployed as a static site to Vercel.
▲
Vercel
Hosting + serverless functions
Hosts the React app and runs the API functions (LinkedIn, Stripe, scheduler). Auto-deploys from GitHub on every push. Free tier covers agency-level usage.
🗄
Supabase
Database + authentication
PostgreSQL database with row-level security — each agency only sees their own data. Handles login/signup, real-time message sync, and all data persistence. Replaces localStorage completely.
in
Unipile
LinkedIn API
The layer that connects LinkedIn accounts via OAuth and executes actions safely — connection requests, messages, profile visits, post engagement. Handles rate limiting and account safety. ~$150/mo.
✉
Resend
Email notifications
Sends reply notifications and daily digests. Simple API, excellent deliverability. Free tier covers 3,000 emails/month which is more than enough for an agency's internal notifications.
$
Stripe
Billing (ready, not active)
Subscription billing infrastructure is fully built — products, checkout, webhook handler. Not activated for internal use but ready to flip on for external clients immediately.
✦
Anthropic Claude
AI across all features
Powers reply intent classification, AI-suggested replies, campaign health diagnosis, client reports, SEO copy reviewer, lead quality scoring, social comment drafting, and AI coach. Claude Sonnet 4.
⏰
cron-job.org
Campaign scheduler
Free service that calls the campaign scheduler every hour. Triggers the Vercel function that checks what messages are due, respects send windows and daily limits, and fires via Unipile.
🐙
GitHub
Version control
Private repo at github.com/mikelanghans/reachflow. Every push to main triggers an automatic Vercel deployment. Full history of every code change.
